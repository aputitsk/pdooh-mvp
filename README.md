# pDOOH

A blockchain-based digital out-of-home (DOOH) advertising platform built on Arc.

## Features

- USDC escrow for advertiser funds
- Private sealed-bid auctions
- Automated on-chain settlement
- Verified smart contract on Arc Testnet
- Browser wallet support (MetaMask / Rabby)
- Built with Next.js, Viem, Solidity, and Foundry
